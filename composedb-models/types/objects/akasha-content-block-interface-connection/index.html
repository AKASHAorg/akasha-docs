AkashaContentBlockInterfaceConnection

A connection to a list of items.

type AkashaContentBlockInterfaceConnection {
  edges: [AkashaContentBlockInterfaceEdge]
  pageInfo: PageInfo!
}

Fields​

AkashaContentBlockInterfaceConnection.edges ● [AkashaContentBlockInterfaceEdge] list object​

A list of edges.

AkashaContentBlockInterfaceConnection.pageInfo ● PageInfo! non-null object​

Information to aid in pagination.

Returned By​

Member Of​
